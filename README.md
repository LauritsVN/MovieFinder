**MovieFinder – Intelligent Filmanbefalinger med AI**
MovieFinder er mit personlige projekt, som er bygget i frustration over at bruge 20 minutter på at vælge en film. I stedet for at scrolle gennem mange lister, ville jeg lave en app, hvor man kan skrive hvad man er i humør til og få et svar med det samme

__Status:__ Under udvikling. Jeg arbejder i øjeblikket på at færdiggøre AI-logikken og finpudse brugerfladen.

**Konceptet**

Applikationen hjælper brugere med at finde den perfekte film ved at kombinere traditionelle søgeparametre (genrer, tidligere sete film) med naturligt sprog

I stedet for blot at klikke på kategorier, kan man f.eks. skrive: "Jeg vil gerne se en dyster sci-fi film med et twist, der minder om Inception", hvorefter systemet kommer med kvalificerede bud.

**Tech Stack**

Frontend: React

Backend: C# (.NET Core)

API-integrationer: * TMDB API: Til hentning af data, billeder og filmdetaljer.

LLM API (Groq): Til at analysere brugerens tekstønsker og generere intelligente forslag.

Sprog: JavaScript og C#.
